## 📝 Description

Azad Ply & Furnitures is a comprehensive web platform built to showcase and sell hardware and furniture products online. The application features a dynamic product catalog, a shopping cart, a contact/inquiry form, and an integrated admin interface for managing the platform.

## ✨ Features

- **Product Catalog**: View a wide variety of hardware, plywood, and furniture products.
- **Shopping Cart**: Users can add products to their cart and checkout via WhatsApp integration.
- **Contact & Inquiries**: Functional email inquiry form to get in touch with the store owners.
- **Admin Dashboard**: Manage products and view store analytics.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Email Service**: Nodemailer

## 🏁 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- [MongoDB](https://www.mongodb.com/) (Local installation or MongoDB Atlas URI)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Shaheen-git767/Azad-Ply.git
   ```

2. Navigate to the project directory:
   ```sh
   cd Azad-Ply
   ```

3. Install NPM packages:
   ```sh
   npm install
   ```

4. Create a `.env` file in the root directory and configure your environment variables. Example:
   ```env
   PORT=3000
   MONGODB_URI=your_mongodb_connection_string
   EMAIL_USER=your_email@example.com
   EMAIL_PASS=your_email_app_password
   ```

5. Start the development server:
   ```sh
   npm start
   ```

6. Open your browser and visit: `http://localhost:3000`
